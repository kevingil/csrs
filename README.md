# CSRS

A first-person shooter built with Rust and Bevy.

## Development & Build

### Prerequisites

- **Rust nightly** (automatically selected via `rust-toolchain.toml`)

### Run

```bash
cargo run
```

### Build Release

```bash
cargo build --release
```

> The project uses Bevy's `dynamic_linking` feature for faster dev compile times. For distribution, remove this feature from `Cargo.toml`.
